# checker & resolver

* 開発言語: C++
* 機能: 1.チェッカー，2.コスト計算，3.コスト削減のための再ルーティング

## 入出力

* 問題ファイル
* 解答ファイル(再ルーティング前)
* 解答ファイル(再ルーティング後) ※任意

## 1.チェッカー

* 解答ファイル(再ルーティング前)が解答ルールを満たしているか確認
* branch error: ラインの分岐が存在
* floating error: 端点に接続しない数字が存在

## 2.コスト計算

* 要オプション(-c)
* 線長の合計と線が曲った回数を導出

## 3.再ルーティング

* 要オプション(-r)
* 1ラインずつ引きはがして再ルーティング
* 曲がる回数が少なくなるようなルーティングを選択
